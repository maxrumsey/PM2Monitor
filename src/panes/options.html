Help

  • Host

    The host of the remote server should go here. Eg: localhost, 192.168.0.1, vps.example.com

  • Username

    The SSH username should go here.

  • KeyFile Location

    The location of the SSH key-file on your machine should go here.

  • Port

    The SSH port for the remote server should go here. Default: 22

  • Server Folder

    This should be the path to the server driver you installed on your remote server.
